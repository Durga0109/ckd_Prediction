## How to Use These Samples

1. Launch the Streamlit app:
   ```
   streamlit run app.py
   ```

2. In the sidebar, you can either:
   - Enter the values manually for each parameter
   - Use the "Load Example Patient" dropdown to select a sample patient

3. If entering manually, refer to the examples above for realistic values.

4. Click "Predict" to get the CKD prediction, stage, and recommendations.

## Key Patterns to Notice

### CKD Patients Often Have:
- Higher serum creatinine (>1.4)
- Lower hemoglobin (<12)
- Higher blood urea (>40)
- Presence of albumin in urine (>0)
- Hypertension and/or diabetes
- Abnormal RBC/pus cells
- Lower packed cell volume

### Non-CKD Patients Often Have:
- Normal serum creatinine (0.5-1.2)
- Normal hemoglobin (>13)
- Lower blood urea (<40)
- No albumin in urine (0)
- No hypertension or diabetes
- Normal RBC/pus cells
- Normal packed cell volume (>40)

These patterns can help you understand how different factors influence the prediction.
